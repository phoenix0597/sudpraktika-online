# Сырые данные рыночного замера ЗоПП

Источник: Yandex Wordstat API (GA), `scripts/wordstat_fetch.py` (breadth-режим),
88 seed-фраз по всем вертикалям ЗоПП. Замер 2026-06-24. Прервано часовым лимитом
(100 запросов/час) на 68-й фразе; 20 фраз добираются позже (возобновляемо).

Этот файл фиксирует данные в памяти проекта (не в `output/`, который исключён из
git), чтобы обеспечить консистентность между сессиями. Сырой CSV дополнительно
хранится в `output/wordstat_market_sweep.csv`.

## Что значат столбцы

- **Спрос/мес** — суммарный спрос кластера = сумма частот seed-фразы + всех
  родственных long-tail запросов из GetTop. Верхняя оценка (кластеры
  пересекаются между собой, поэтому сумма не строго аддитивна).
- **Seed (total_shows)** — частота только самой seed-фразы.
