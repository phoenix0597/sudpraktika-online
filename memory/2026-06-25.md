## Анти-галлюцинация — двойная защита (критическое правило)
Пользователь поймал галлюцинации ссылок — главную угрозу для юридического сайта. Решение:
1. Промпт с явным запретом генерировать ссылки не из текста акта (`scripts/prompt_practice_analysis.txt`).
2. Программный верификатор `scripts/verify_citations.py` — сверяет каждую ссылку с текстом акта.
Эмпирически: v1 промпт = 4 галлюцинации, v2 с запретом = 0. Качество анализа сохранено.
Правило закреплено в `technical-specification-mvp.md` и `assistant-memory.md`.

## Интеграция Antigravity (гибридный путь извлечения)
Пользователь предложил альтернативу DeepSeek для масштабирования — ИИ-агент Antigravity (Gemini-3.5-flash) с щедрыми лимитами (1500+ запросов/сутки при PRO-подписке), десктоп + CLI. Спроектирована интеграция через «почтовый ящик» (контракты на обмен файлами, без жёсткой связки):
- `data/inbox/_TASK.md` — единая инструкция для любого агента (роль, 2 задачи, анти-галлюцинация, стиль, формат).
- `data/inbox/_queue.json` — очередь актов (pending → done).
- `scripts/queue_for_agent.py` — наполняет очередь из data/raw_acts/.
- Вывод любого агента гоняется через `verify_citations.py`.
- Режимы: ручной запуск (MVP) + автоматический CLI.
Закреплено в ТЗ, assistant-memory.md, current-work-queue.md.
