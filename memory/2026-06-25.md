Пользователь поймал галлюцинации ссылок — главную угрозу для юридического сайта. Решение:
1. Промпт с явным запретом генерировать ссылки не из текста акта (`scripts/prompt_practice_analysis.txt`).
2. Программный верификатор `scripts/verify_citations.py` — сверяет каждую ссылку с текстом акта.
Эмпирически: v1 промпт = 4 галлюцинации, v2 с запретом = 0. Качество анализа сохранено.
Правило закреплено в `technical-specification-mvp.md` и `assistant-memory.md`.
