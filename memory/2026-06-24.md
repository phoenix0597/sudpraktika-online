## Фаза 0, Шаг 1 — рыночный замер и вердикт жизнеспособности
Широкий sweep 88 фраз по всем вертикалям ЗоПП (breadth-режим). Объём спроса (с учётом long-tail) ≈434K показов/мес по 68 фразам. Лидеры: маркетплейсы (112K: Озон 60K, ВБ 47K), процессуальные темы (107K: претензия-образец 30K, статья ЗоЗПП 32K), ДДУ/жильё (59K), финансы/страховки (44K). **Вердикт: GO** — спрос большой, окно для входа есть. Нашли часовой лимит Wordstat API (100/час). Исправлена ошибка занижения спроса (ранее считали только seed-фразу) — агрегатор теперь суммирует весь long-tail кластера.

**Решения пользователя 2026-06-24:**
1. Пересмотр пилота → маркетплейсы + процессуальные (по данным замера).
2. РСЯ — основной канал монетизации; цель — оптимистичный прогноз (~54K ₽/мес при ~1 200 уник/день).

## Внешнее ревью ТЗ/стратегии — точечные правки
Получено компетентное ревью от другого ИИ-агента. Принято ~80%, скорректирован framing и защищён holding как ядро продукта. Внесены правки:

- **Источники:** сбор с официальных страниц судов (sudrf.ru) как первичный источник; sudact.ru/rospravosudie.com — справочно. Акт как официальный документ не охраняется АП, забор поодиночке выводит из серой зоны права изготовителя базы (ст. 1334 ГК РФ). Юридическую оценку даёт пользователь-юрист.
- **Экономика РСЯ:** переход с CPC на RPM-модель с тремя сценариями; подключение РСЯ после появления трафика.
- **Дедупликация:** по идентичности дела (номер + суд + дата + инстанция) + текстовый хэш; не по фабуле.
- **SEO-разметка:** убран HowTo; снижены ожидания от FAQ rich-results; тип схемы WebSite/Organization/Article вместо LegalService.
- **Плашка верификации:** честная перстраничная маркировка вместо blanket-плашки «верифицировано юристом».
- **Вехи трафика:** верхняя граница скорректирована до ~20K/день (50K математически невозможны из пула спроса 5–10K запросов/день).
- **Стек:** платёжный шлюз перенесён в пост-MVP (осиротел после переноса DOCX); убран блок «что изменилось бы, если…» (контрфактичен, риск + сложность); термин «прецедент» → «позиция суда / судебная практика».
- **Holding сохранён** как дифференциатор/moat — ревью предлагало его убрать, отклонено.
